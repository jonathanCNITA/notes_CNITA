### Getting started
```html
<!DOCTYPE html>
<html lang="fr">
<head>
	<meta charset="UTF-8">
	<title>Page title here</title>
	<link rel="stylesheet" type="text/css" href="css/style.css"/>
	<script type="text/javascript" src="script.js"></script>
</head>
<body>
	<h1>Getting started page</h1>
</body>
</html>
```
